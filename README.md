# Kitap Kütüphanesi

Okuma deneyimini sade ve keyifli hale getirmeyi amaçlayan, farklı türlerden kitapları bir araya getiren çevrim içi bir kitap kütüphanesidir.

==> Özellikler
- Kitap koleksiyonunu görüntüleme
- Kitap sayfalarına göz atma
- Favori kitapları kaydetme
- Hızlı arama ve filtreleme

==> Kullanılan Teknolojiler
- HTML
- CSS
- Bootstrap
- JavaScript
- JSON

==> Canlı Demo
Proje GitHub Pages üzerinden erişime açılmıştır.
